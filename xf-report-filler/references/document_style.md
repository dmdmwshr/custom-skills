# Word 公文格式

本文件记录本 skill 新生成 Word 公文类材料的默认格式。外部模板已有固定样式时，以外部模板为准；脚本新建 `.docx` 时必须按本规则设置。

## 默认格式

- 标题：2 号 `方正小标宋_GBK`，居中。
- 正文：3 号 `方正仿宋_GBK`，一般每页 22 行、每行 28 个字。
- 一级标题：3 号 `方正黑体_GBK`。
- 二级标题：3 号 `方正楷体_GBK`。
- 三级、四级标题：3 号 `方正仿宋_GBK`。
- 生成 `.docx` 时要同时设置中文 eastAsia 字体、字号、A4 页面、页边距和文档网格，不能只改可见 run 的西文字体。

## 年度产品问题汇总

- 封面标题使用 2 号方正小标宋。
- 大队作为一级标题，月份作为二级标题，用于 Word 左侧导航。
- 案卷信息和问题正文均使用 3 号方正仿宋；案卷信息可加粗但不进入标题层级。
- 不生成可见目录页，不插入 TOC 字段。
