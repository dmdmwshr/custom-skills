# 数据源：联网基础信息考评截图

## 文件用途

联网基础信息考评截图是联网监测问题和案卷分数的数据源之一，不对外发送。

## 依赖的数据源和模板

- 优先读取成绩月份巡查目录根层 `YYYY年M月基础信息考评截图（不发）.xls`。
- 根层不存在时，可在 `YYYY年M月联网监测基础信息考评明细表` 等明细文件夹内查找 `*基础信息考评截图*.xls`。
- 下游模板：T04、T06、T07，以及根层 R01 的联网系统核查行。

## 数据源解析

- 按大队 sheet 或表格区域读取案卷分数、备注和问题描述。
- 备注中的 `消防机构联系人`、`联系人` 等伪联系人作为问题标签，不当作人员姓名。
- CAD/PDF 类文本在公开输出前统一转换口径。

## 生成规则

- 联网问题进入月通报联网段、个人执法统计联网列、个案成绩表和根层科室表联网系统核查。
- CAD 类问题输出为“缺点位图”。
- PDF 类问题输出为“缺火灾防控图”。
- 已经规范为点位图或火灾防控图的文本保持规范口径。

## 不写入内容

- 不修改源截图表。
- 不把伪联系人写入个人执法统计表。
- 不在公开材料中写 CAD、PDF、缺CAD图、缺PDF。

## Warning/Blocker

- 找不到基础信息考评截图：blocker。
- 伪联系人被识别：warning，不阻塞。
- 必要 sheet 或分数列缺失：blocker。

## 验收点

- 月通报和科室表只出现“缺点位图”“缺火灾防控图”。
- 个人执法统计表不写入伪联系人。
- dry-run 输出 sheet、行列和问题定位。
