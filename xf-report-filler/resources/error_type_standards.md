# 产品案卷错误类型与评分归类标准

本文件用于月度产品成绩登记时，把巡查底册中的自然语言错误归类到《消防产品档案质量明细表》的一般要素。机器可读的正式分值仍以 `scoring_rules.js` 和 `scoring_rules.json` 为准；本文件只固化 2-5 月登记中反复出现的错误表述和默认归类口径。

## 总体口径

1. 底册扣分行已明确写出 `3.x` 和分值时，优先使用底册分值，不重新推断。
2. 同一一般要素下出现多个错误时，合并到同一行，扣分相加，但不超过评分规则中的该项上限。
3. 只有错误描述、没有明确扣分行时，不自动扣分；先列入待核对。
4. 大队本月无产品案卷时，产品全链路记 `0` 分；不按空白案卷记 `10` 分。

## 错误类型映射

| 归类 | 对应一般要素 | 常见错误表述 | 常见扣分 |
| --- | --- | --- | --- |
| 检查记录填写问题 | 3.1 消防产品监督检查记录 | 备注为空、记录不规范、记载缺失、空白处未斜杠划除、检查记录文书模糊 | 0.1-0.2 |
| 现场判定通知书问题 | 3.2 消防产品现场检查判定不合格通知书 | 具体问题描述不清晰、产品名称/型号/生产企业填写不全、与检验报告或现场照片不一致 | 0.2-0.5 |
| 抽样单问题 | 3.3 消防产品质量监督抽查抽样单 | 抽样单填写不规范、备注材料数量未填写、准入证书或检验报告未填写 | 0.1-0.5 |
| 检验结果通知书问题 | 3.4 消防产品质量检验结果通知书 | 检验结果通知书缺审批表、内容填写不全、与其他材料描述不一致 | 0.1-0.2 |
| 复检材料问题 | 3.5-3.7 复检相关文书 | 复检申请表缺签名盖章、复检受理凭证缺失、复检结果通知书填写不规范 | 0.1-0.5 |
| 通报函问题 | 3.8/3.9 通报涉嫌违法生产销售或生产不合格消防产品案件函 | 通报函缺失、呈请审批表未上传、办案部门意见未签字 | 0.1-1.0 |
| 后续处理文书问题 | 3.10 责令限期改正通知书、复查记录、处罚决定等 | 责令限期改正通知书具体问题未写规格型号、生产企业、不合格通知书编号；限改审批表缺失；相关文书未入卷 | 0.1-1.0 |
| 照片证据问题 | 3.11 消防产品样品照片等证据材料 | 缺现场照片、复查照片不全、照片缺拍摄人签名/时间/文字说明、照片概貌缺失 | 0.1-0.5 |
| 送达回证问题 | 3.12 送达回证 | 送达回证时间不全、送达文书名称/文号/受送达人错误、超期送达 | 0.1-0.5 |
| 非特定文书规范问题 | 3.13 非特定文书规范性问题 | 缺授权委托书、缺法人证明、缺营业执照、审批表未上传纸质盖章件、材料上传错误、文书缺签字 | 0.1-0.5 |
| 归档问题 | 3.14 立卷归档的其他规范性问题 | 卷内目录、页号、备考表、装订顺序、题名编号等归档格式问题 | 0.1-0.2 |
| 系统应用问题 | 3.15 消防监督管理系统应用 | 系统无信息、系统文书与纸质案卷不一致、未按规定流转或上传 | 0.5-1.0 |

## 月度登记输出要求

- 产品通报：写“大队 + 承办人 + 被检查单位 + 错误概述”；无案件大队写明“本月无消防产品监督检查案卷，产品成绩记0分”。
- 科室月考核：产品行保留分项编号换行；联网行按 10 个抽查单位依序列出。
- 个人执法统计：产品只填立卷人；联网按联系人逐条填入，不把大队平均分填给个人。
- 个案成绩汇总：产品列取产品总表分数；联网列取 `联网监测统计表.xls` 平均分。
